## 0.2.3.3: About fix + Logs access

Patch release fixing macOS menubar About and adding agent log discovery across menubar, macOS Settings, and the web dashboard.

### Fixes

- **macOS menubar:** **About llm-swarm-router** opens reliably via a custom SwiftUI window with `NSApp.activate` (accessory menubar apps cannot depend on `orderFrontStandardAboutPanel`)

### Features

- **Logs API:** `GET /netllm/v1/logs?tail=N` (loopback admin, read-only) returns log paths, size, and tail lines
- **Web dashboard:** **Logs** tab with refresh, copy path, and 10s poll while active
- **macOS Settings:** **Logs** sidebar tab with tail view, Refresh, Reveal in Finder, Open in Console
- **macOS menubar:** **Open Log File** and **Open Log Folder** shortcuts

### Install / upgrade

- **macOS:** [v0.2.3.3](https://github.com/matthewdcage/llm-swarm-router/releases/tag/v0.2.3.3) — menubar **Updates** menu or web dashboard at `/ui/`
- **Source:** `git pull && uv sync`
