## 0.2.2.1: macOS menubar startup fix

Patch release for the native macOS app.

### Fixes

- **Menubar agent status:** The app no longer reports "running" when `/health` succeeds on a *different* `netllm serve` already bound to port 11400 (e.g. a stale terminal session). It now requires its own supervised child process to be alive before marking the agent running.

### Install

- macOS: download `llm-swarm-router.dmg` from [GitHub Releases](https://github.com/matthewdcage/llm-swarm-router/releases/tag/v0.2.2.1)
- Linux/Windows alpha packages unchanged in behavior; rebuilt at 0.2.2.1 for version alignment
