## 0.2.3.2: Auto-update (macOS in-app + web dashboard)

Patch release adding GitHub release checking, verified DMG downloads, and in-app install for Applications installs.

### Features

- **Web dashboard:** `/netllm/v1/version` and `/netllm/v1/update/check` proxy; Status tab shows version, install method, Updates card with download links and hourly poll
- **macOS menubar:** `UpdateController` checks GitHub, downloads and verifies SHA256 sidecars, one-click install via bundled `macos-app-install.sh` (`--in-app-update`, `--wait-for-pid`, `--cache-cleanup`)
- **Config:** `[ui] check_for_updates_automatically = true` (disable for manual-only checks)
- **Release CI:** SHA256 sidecars for DMG/deb/rpm/zip; `NETLLM_VERSION` passed to macOS build

### Fixes

- **Bundled installer:** `mount-dmg.sh` resolves from `Scripts/` inside `.app`; macOS BSD `sed` port parse for post-install health verify

### Install / upgrade

- **macOS:** [v0.2.3.2](https://github.com/matthewdcage/llm-swarm-router/releases/tag/v0.2.3.2) — menubar **Updates** menu or web dashboard at `/ui/`
- **Source:** `git pull && uv sync`
