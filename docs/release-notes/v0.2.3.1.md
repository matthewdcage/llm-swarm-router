## 0.2.3.1: macOS clean DMG upgrade

Patch release for the native macOS app. Bundles the upgrade installer scripts and documents the fix for post-upgrade `/ui/` 404s.

### Fixes

- **Clean DMG upgrades:** `macos-app-install.sh` quits the menubar app, stops Homebrew `netllm` if running, clears orphaned agents on the listen port, replaces `/Applications/llm-swarm-router.app`, and verifies `GET /ui/` returns HTTP 200
- **Bundled in app:** `Contents/Resources/Scripts/macos-app-install.sh` and `mount-dmg.sh` for upgrades without a repo clone

### Install / upgrade

- **macOS:** `llm-swarm-router.dmg` from [GitHub Releases](https://github.com/matthewdcage/llm-swarm-router/releases/tag/v0.2.3.1)
- **Upgrade from downloaded DMG:**

```bash
/Applications/llm-swarm-router.app/Contents/Resources/Scripts/macos-app-install.sh \
  --dmg ~/Downloads/llm-swarm-router.dmg
```

- **Linux/Windows:** Rebuilt at 0.2.3.1 for version alignment; same **alpha** behavior as v0.2.3
