## 0.2.2 beta: Linux and Windows packages

First release with Linux and Windows beta install assets attached alongside the macOS DMG.

### New assets

- `netllm_0.2.2_amd64.deb`: Linux (systemd user service)
- `netllm-0.2.2-*.rpm`: Linux RPM
- `netllm-0.2.2-windows-x64.zip`: Windows portable + service installer
- `netllm.yaml`: Winget manifest with SHA256 (submit to winget-pkgs separately)

### Install

- Linux: [docs/linux-install.md](https://github.com/matthewdcage/llm-swarm-router/blob/main/docs/linux-install.md)
- Windows: [docs/windows-install.md](https://github.com/matthewdcage/llm-swarm-router/blob/main/docs/windows-install.md)
- macOS: DMG (stable): unchanged

### Beta notes

- Linux/Windows use the web dashboard at http://127.0.0.1:11400/ui/ (no native tray)
- Windows mDNS may require firewall rules or static peers
- Source install path unchanged: `uv sync` + `./netllm serve`

### Fixes in this release

- CI green on ubuntu-latest and windows-latest
- Packaging uses `uv sync --no-dev` (compatible with current uv)
- Unified release workflow builds all platforms on publish
